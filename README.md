# 🎂 Happy Birthday Paridhi! 💕

A beautiful, interactive birthday website created with React, Vite, Tailwind CSS, and Framer Motion.

## 🌟 Features

- **Home Page**: Stunning welcome with birthday wishes and animated hearts
- **Love Letter**: Beautiful handwritten-style love letter with elegant animations
- **Cake Ceremony**: Interactive candle blowing and cake cutting experience
- **Star Page**: Special dedication connecting Sirius star to "ParidhiStar" with space for photos
- **Gallery**: Photo gallery placeholder for your memories together
- **Memories**: Interactive memory cards with beautiful transitions

## 🛠️ Tech Stack

- **React 18** - UI framework
- **Vite** - Build tool and dev server
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations
- **Google Fonts** - Handwriting fonts (Dancing Script, Pacifico, Great Vibes, Satisfy)

## 📦 Installation

```bash
npm install
```

## 🚀 Running the Project

```bash
npm run dev
```

## 🏗️ Building for Production

```bash
npm run build
```

## 🎨 Customization

### Adding Real Photos

1. **Sirius Star Photo**: Replace the placeholder in `src/components/StarPage.jsx`
2. **Paridhi's Photo**: Replace the placeholder in `src/components/StarPage.jsx`
3. **Gallery Photos**: Add your real photos in `src/components/Gallery.jsx`

### Adding Birthday Song

Add an audio file and uncomment the audio code in `src/components/CakeCeremony.jsx`

### Customizing Content

All text content can be easily customized in the component files:
- `src/components/Home.jsx` - Home page wishes
- `src/components/LoveLetter.jsx` - Love letter content
- `src/components/StarPage.jsx` - Star dedication
- `src/components/Memories.jsx` - Memory cards
